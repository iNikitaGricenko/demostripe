# DemoStripe ``Portfolio project``
Java backend restfull application with strapi connection

# Requirements:
- Java 17 or above
- Apache Maven 3.6.3 or above

# Technologies

## Backend
* Spring Boot
* Spring Web
* Spring Validation
* Swagger
* Redis Caching
* JPA
* AspectJ

### Databases
#### SQL
* PostgreSQL

#### NoSQL / Cache
* Redis

#### Migrations
* Flyway

### Code Write
* Lombok
* Mapstruct

### Services
* Github actions
* Docker & Docker-compose
